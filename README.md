# 🌀 Naruto Mobile App

A Naruto-themed mobile application built using **React Native (Expo)**.  
The app fetches real-time data from the Dattebayo API and provides an interactive experience to explore characters, clans, villages, and more.

---

## 🚀 Features

- 📱 Browse Characters, Clans, Villages, Teams, etc.
- 🔍 Search functionality for quick filtering
- 🧭 Drawer + Stack Navigation
- 🔗 Clan → Members → Character Detail navigation
- ⚡ Real-time API integration
- 🎨 Clean UI with reusable components

---

## 🛠 Tech Stack

- React Native (Expo)
- TypeScript
- React Navigation (Drawer + Stack)
- Dattebayo API
- Custom Theme System

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/your-username/naruto-app.git

# Navigate to project folder
cd naruto-app

# Install dependencies
npm install

# Start the app
npx expo start
```
